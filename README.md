
# React JSON Data Generator App

This project was built using react and hosted on netlify. You can generate fake data in JSON format.


## Demo

https://url-shrinker.netlify.app

## NPM packages used
- react
- materialui
- downloadjs
- faker


## Deployment

To deploy this project run

```bash
  npm start
```
